# 🥚 Tamamon

Un juego **idle/clicker** de monstruo digital al estilo Digivice, con estética Game Boy. Empieza con un huevo, hazlo eclosionar a base de toques, alimenta y entrena a tu criatura, compra mejoras y evoluciona a través de 6 etapas: HUEVO → BEBÉ → NIÑO → ADULTO → PERFECTO → MEGA.

## ✨ Características

- 🥚 6 etapas de evolución, desde HUEVO hasta MEGA.
- 👆 Mecánica idle/clicker: toca para hacer crecer a tu Tamamon.
- 🍖 Acciones de cuidado: alimentar (FEED) y entrenar (TRAIN).
- ⬆️ Tienda de mejoras (auto-tap, multiplicadores, etc.) con costes escalables.
- 🎨 Estética Digivice + Game Boy con marco de dispositivo y pantalla pixelada.
- 🔔 Notificaciones tipo toast para los eventos del juego.

## 🚀 Cómo jugar / ejecutar

No necesita build ni dependencias locales (PixiJS se carga por CDN). Abre el archivo en tu navegador:

```
open index.html
```

O sírvelo localmente:

```
python3 -m http.server 8000
# luego abre http://localhost:8000
```

## 🎮 Controles

- 👆 Toca/clica la pantalla del dispositivo para hacer crecer a tu Tamamon.
- 🔘 **A**: FEED (alimentar) · **B**: TRAIN (entrenar) · **C**: UPGRADES (mejoras).

## 🛠️ Tecnología

- [PixiJS](https://pixijs.com/) v7 (cargado por CDN) para el renderizado.
- JavaScript puro, sin build.
- Sprites de criaturas definidos en `creatures.js`.
